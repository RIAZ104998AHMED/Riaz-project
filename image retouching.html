How to use:

  1. Upload an image using the file selector
  2. Choose a retouching tool from the dropdown
  3. Adjust brush size and intensity as needed
  4. Click and drag on the image to apply retouching
  5. Use Undo/Redo to correct mistakes
  6. Download your retouched image when finished

Tool Tips:

Blur: Softens selected areas

Clone Stamp: Click to set source point, then paint to clone

Lighten/Darken: Adjusts brightness in selected areas

Sharpen: Enhances edges in selected areas

Red Eye Reduction: Fixes red-eye in portraits
